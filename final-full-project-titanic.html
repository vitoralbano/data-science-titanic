def create_figure(title):
    """
        Criar uma figure com titulo posicionado
        
        Args:
            title (String): Valor a ser preenchido no ttulo do grfico 
            
        Return:
            (plt.Figure) Retorna uma Figure para adicionar grficos (Plots)
    """
    fig = plt.figure(title)
    fig.suptitle(title, fontsize='x-large', horizontalalignment='center', verticalalignment='bottom')
    plt.subplots_adjust(top=.9)
    plt.tight_layout()
    fig.set_figwidth(20)
    
    return fig

def get_survivors():
    """
        Exibir tabela com contagem de sobreviventes e proporo
        
        Return (dataframe)
    """
    # Definio do dataframe de propores
    df_survivors = df['sobreviveu'].value_counts().reset_index()
    df_survivors.rename(columns={'index': 'sobreviveu', 'sobreviveu': 'total'}, inplace=True)
    df_survivors['proporcao'] = round(df_survivors['total'].apply(lambda value: value / df_survivors['total'].sum()) * 100, 2)
    
    return df_survivors

def plot_survivors():
    """
        Exibir pie plot de sobreviventes
    """
    df_survivors = get_survivors()
    fig = create_figure('')
    
    # pie plot
    ax_pie = fig.add_subplot(121)
    
    values = df_survivors['proporcao'] * 100
    explode = np.full(df_survivors.shape[0], 0.05)
    labels = df_survivors['sobreviveu'].replace(['Sim','No'],['Sobreviveu', 'No sobreviveu'])

    ax_pie.axis('equal')
    ax_pie.pie(
        values, 
        explode=explode, 
        labels=labels, 
        autopct='%1.1f%%', 
        shadow=True,
        startangle=90,
    );
    
    ax_pie.set_title('SOBREVIVENTES')
    
    # table plot
    ax_table = fig.add_subplot(122)
    table = ax_table.table(
        cellText = df_survivors.values,
        colLabels = ['SOBREVIVEU', 'TOTAL', 'PROPORO'],
        loc='center',        
    )
    ax_table.axis('off')
    ax_table.set_title('TABELA DE SOBREVIVENTES')
    table.auto_set_font_size(False)
    table.set_fontsize(13)
    
plot_survivors()

def plot_children_balance():
    """
        Grfico de crianas sobreviventes e no sobreviventes
    """
    df_children = df.query('categoria_idade == "Criana"')['sobreviveu'].value_counts(normalize=True).to_frame().reset_index()
    df_children.rename(columns={'index': 'sobreviveu', 'sobreviveu': 'percentual'}, inplace=True)
    df_children['percentual'] = round(df_children['percentual'] * 100, 2)
    
    ax = sns.barplot(
        data = df_children,
        x = 'sobreviveu',
        y = 'percentual',
        palette = 'Blues_d',
    )
    
    plt.title('SOBREVIVENTES: CRIANAS')
    plt.xlabel('Sobreviveu')
    plt.ylabel('Percentual')
    
    
plot_children_balance()

Apresentação¶

Mesmo aps um sculo de seu naufrgio, que ocorreu em 1912, o Titanic considerado um dos maiores desastres martimos em tempos de paz. Sua histria rendeu livros, filmes e diversos documentarios que buscam explorar e levar informaes e curiosidades a seu respeito ao pblico. Nesse mesmo sentido, esse projeto tem como objetivo explorar o conjunto de dados do Titanic e tentar responder uma srie de perguntas pertinentes. O arquivo est disponvel no formato CSV atravs do link.

No primeiro instante, ao carregar o conjunto de dados, foi necessario avaliar suas caractersticas como seu esquema de organizao, consistncia dos dados e a necessidade de possveis correes ou adaptaes que pudessem contribuir na manipulao, pesquisa e anlise. Em tempo, o conjunto de dados apresentava a seguinte estrutura:

  • PassengerId: Nmeros inteiros, correspondente ao nmero de identificao do passageiro;
  • Survived: Preenchido com nmeros inteiros, 1 para os que sobreviveram e 0 para os no sobreviveram;
  • Pclass: Nmeros inteiros correspondentes ao ordinal da Classe do passageiro, com valores 1, 2 e 3;
  • Name: Nome do passageiro;
  • Sex: Descrio do sexo do passageiro em ingls 'Female' e 'Male';
  • Age: Nmero indicando a idade do passageiro;
  • SibSp: Nmero de irmos e ou conjuge abordo;
  • Parch: Nmero de parentes como pais ou filhos a bordo;
  • Ticket: Texto de identificao da passagem;
  • Fare: Tarifa paga pelo passageiro;
  • Cabin: Texto de identificao da cabine do passageiro;
  • Embarked: Texto indicando o local de embarque do passageiro;

Aps o reconhecimento dos campos, fundamental a sua validao. Buscar por valores nulos e preenche-los com valores pertinentes, encontrar linhas duplicadas, verificar tipos de dados e possveis converes. Neste passo foram aplicadas as seguintes mudanas:

  • Nomes de colunas e valores foram traduzidos para o portugus de modo que ficassem no mesmo idioma da anlise;
  • Valores nulos na coluna 'idade' foram preenchidos com o valor mdio;
  • Passageiros foram classificados por idade como Criana, menores de 15 anos, Jovem, entre 15 e 25, Adulto, entre 25 e 65, e Idoso para maiores de 65 anos;
  • Valores abreviados na coluna 'embarcou' foram substituidos pelo nome correspondente sem abreviao;
  • Passageiros que sobreviveram e no possuiam local de embarque foram removidos do conjunto, considerando que no embarcaram;

Feito isso, foram levantadas perguntas das quais poderiam ser respondidas com o conjunto de dados disponvel, so elas:

  1. Quantos sobreviveram?
  2. Quantos morreram?
  3. Quantas crianas sobreviveram?
  4. Quantas crianas morreram?
  5. Qual a idade mdia entre os sobreviventes?
  6. Qual a idade mdia entre os que morreram?
  7. Qual a relao dos sobreviventes com a classe de ingresso?
  8. Qual o sexo mais relevante entre os sobreviventes?
  9. Qual a relao dos sobreviventes com a classe e sexo?
  10. Qual ponto de embarque recebeu mais passageiros?
  11. Do sexo masculino, qual a idade do sobrevivente mais velho?
  12. Do sexo masculino, qual a idade do sobrevivente mais novo?
  13. Do sexo feminino, qual a idade da sobrevivente mais velha?
  14. Do sexo feminino, qual a idade da sobrevivente mais nova?

Conclusão¶

Ao analisar a amostra de dados de passageiros do Titanic, foi possvel chegar as seguintes concluses.

  • A primeira classe apresentou o maior percentual de sobreviventes, com 62,6%, contra 47,2% da segunda e 24,2% da terceira. Tal diferena pode apontar que houve alguma facilidade ou favorecimento no acesso aos botes salva vidas para os passageiros da primeira classe.
  • Dentre os sobreviventes, 68% so mulheres e 32% homens
  • O grupo de classe e sexo que apresentou maior taxa de sobreviventes foi o feminino da primeira classe, com a relao de 96,7%
  • 59,0% das crianas a bordo foram salvas
  • A idade mdia entre os sobreviventes era de 28,4 anos.

Limitações¶

Inicialmente ao avaliar o conjunto de dados foi detectado que haviam campos como idade, embarcou e cabine, no preenchidos e que potencialmente prejudicariam a anlise. Assim, foram feitos alguns ajustes como:

  • Campos de idade com valor nulo foram preenchidos com a mdia
  • Foi efetuada uma breve anlise sobre os passageiros que apresentavam o campo 'embarcou' como nulo e se haviam sobrevivido, assim, dado que o retorno foi positivo, estes foram removidos do conjunto de dados considerando que estes no embarcaram.

Valores abreviados em local de embarque (embarcou) foram substitudos pelo nome do local sem abreviao.

Foram aplicadas tradues nos nomes das colunas e nos valores dos campos sobreviveu e sexo para auxiliar na criao de legendas dos grficos.

Foi criado um campo de categorizao da idade dos passageiros com o objetivo de mapea-los facilmente e levantar informaes a respeito de cada grupo.
