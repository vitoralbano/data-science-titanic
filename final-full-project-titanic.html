# Declarao da funo para categorizar idade
def age_category(age):
    """
        Categorizar idade
        
        args:
            age (int): Idade a ser categorizada
            
        return (String): Nome da categoria
    """

    # Used as reference:
    # https://www.statcan.gc.ca/eng/concepts/definitions/age2
    if(age < 15):
        return 'Children'
    elif(age < 25):
        return 'Youth'
    elif(age < 65):
        return 'Adult'
    else:
        return 'Senior'

def primeira_analise():
    
    fig, ax = plt.subplots(1, 2)
    fig.suptitle(title, fontsize='x-large', horizontalalignment='center', verticalalignment='bottom')

    # Redimensionar o layout automaticamente
    plt.tight_layout()
    # Ajuste de posicionamento em relao ao topo
    plt.subplots_adjust(top=.9)
    # Definio da largura dos grficos
    fig.set_figwidth(14)

    # Definio do dataframe de propores
    df_proportions = df['Survived'].value_counts(normalize=True).reset_index().rename(columns={'index':'Survived','Survived':'Proportions'})
    df_proportions['Proportions'] = df_proportions['Proportions']

    # Definio dos ttulos
    ax[0].set_title('Survivors count')
    ax[1].set_title('Survivors proportions')

    # Gerar os grficos com seaborn
    sns.countplot(data=df, x='Survived', ax=ax[0], palette='Blues_d')
    sns.barplot(data=df_proportions, x='Survived', y='Proportions', ax=ax[1], palette='Blues_d');

TITANIC - Um breve olhar sobre seu naufrágio¶

Mesmo aps um sculo de seu naufrgio, que ocorreu em 1912, o Titanic considerado um dos maiores desastres martimos em tempos de paz. Sua histria rendeu livros, filmes e diversos documentarios que buscam explorar e levar informaes e curiosidades ao pblico.

Nesse mesmo sentido, com o objetivo de aplicar conhecimentos do curso de Data Science do Udacity, este projeto ir explorar um conjunto de dados disponibilizado na etapa final atravs do link.

Perguntas¶

Perguntas elaboradas ao conhecer o conjunto de dados fornecido:

1. Quais são os números de sobreviventes?¶

A primeira pergunta buscou conhecer o tamanho da tragdia. Quantas pessoas sobreviveram, quantas vidas foram perdidas, qual o tamanho da amostra. Nesse sentido busquei a descrio do conjunto de dados e a exibio de grafcos de frequncia e proporo para sobreviventes e mortes.

2. Quais são as características do maior grupo de sobreviventes?¶

Com o propsito de conhecer melhor sobre os sobreviventes e mortos os dividi em grupos de gnero, classe, local de embarque, com o propsito de encontrar qual grupo apresentava maior taxa de sobrevivncia. Explorando suas descries, normalizaes e grficos de frequncia.

3. Qual é a descrição de idade dos passageiros?¶

Dentre a amostra fornecida, qual a idade? Como poderia classifica-los, quais eram as faixas etrias, qual a idade mais presente entre eles? Qual era a idade da pessoa mais velha, e a mais nova? Para isso, os dados foram agrupados por classe, faixa etria e gnero e posteriormente explora-los.

4. Quantas crianças estavam a bordo? Quantas sobrevieram e quantas morreram?¶

Crianas, quantas eram? Sobreviveram? Qual o balano? Utilizando tambm da classificao de faixa etaria, foi possvel mapear as crianas, idades e o balano de sobreviventes.

5. Em qual classe haviam mais familiares? (Irmãos, conjuges, pais, filhos)¶

Aps explorar os dados sobre as crianas, busquei relacionar quais dados poderiam relacionar famlias a bordo, irmos ou conjuges e pais ou filhos. Contando e normalizando informaes sobre as pessoas que apresentavam ao menos um relacionamento familiar nos campos disponveis.

Limpeza de dados¶

O primeiro passo foi listar os tipos de dados utilizados e se havia campos nulos. Feito isso, foi constatado que haviam campos nulos em Idade (Age), Local de embarcao (Embarked) e Identificao da cabine (Cabin). Assim:

  • Os campos com idade nula foram preenchidos com o valor da mdia;
  • Foi feita uma pesquisa pelos passageiros que no possuiam local de embarao e consultado seu status de sobrevivente ou no e foi constatado que nenhum havia morrido. Nesse sentido, foi considerado de que no estavam a bordo e seus registros removidos;
  • Cabin foi mantido nulo, pois diante do quadro de informaes disponveis, no apresentava potencial informativo, poderia ser descartado;
  • As abreviaes dos locais de embarao foram substituidos pelo nome completo;
  • Foram criadas categorias para faixa etria;
